		type name;
		type name[SIZE];
type - тип элементов массива. 
	   Все элементы массива могут быть только одного 
	   определенного типа.
name - имя массива. Для именования массивов используются
	   такие же идентификаторы, как и для именования переменных.
SIZE - количество элементов массива, может быть задано лишь 
	   целочисленным константным значением.
